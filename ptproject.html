ITA

Un progetto per il corso di networking.

Questo file packet tracer rappresenta tre reti di biblioteche universitarie. Tutte le reti hanno diverse vlan per gli studenti e per i bibliotecari e un server che funge da DNS/DHCP server. EtherChannel è impiegato su switch collegati con cavi ethernet crossed-over per aumentare la velocità tra wap distanti. I router usano il routing dinamico per instradare i pacchetti di livello 3 su cavi seriali.

ENG

A packet tracer project for a netwroking class

This packet tracer file represents three university libraries' networks. All networks have different vlans for students and librarians and a DNS/DHCP server. EtherChannel improves connection speed between two switches (connected by crossed-over ethernet cables) that connect further wireless access points. Routers use dynamic routing to route layer 3 traffic on serial cables.
